# calculators
## Different financial calculators

[*] SIP Calculator: Estimates the maturity amount of monthly SIPs based on expected returns, amount, and tenure.
[*] Step Up/Top-up SIP Calculator: Calculates returns when the investment amount is increased annually.
[*] Future value Calculator: Estimates future value of money
[*] Simple interest calculator
[*] Compound interest calculator
[*] Mutual Fund Returns Calculator: A general tool to calculate historical or expected returns (CAGR/XIRR).
[*] Goal Planner: Calculate monthly savings for a target goal.
[*] Retirement Corpus: Plan how much you need to retire comfortably.
[ ] Lumpsum Calculator: Projects the future value of a single, one-time investment.
[ ] SWP (Systematic Withdrawal Plan) Calculator: Helps determine how much money can be withdrawn regularly while leaving the remaining amount invested.
[ ] SSY (Sukanya Samriddhi Yojana Calculator): Calculate returns for Sukanya Samriddhi Yojana Calculator as per your investment.
[ ] PPF (Public Provident Fund): Calculate your returns on Public Provident Fund(PPF).
[ ] EPF (Employee's Provident Fund): Calculate returns for your Employee's Provident Fund(EPF).
[ ] FD: Check returns on your fixed deposits(FD's).
[ ] RD: Check returns on your recurring deposit(RD's).
[ ] NPS: Calculate returns on National Pension Scheme(NPS).
[ ] EMI Calculator
[ ] STP (Systematic Transfer Plan) Calculator: Estimates growth when transferring money systematically between funds.
[ ] Brokerage/MTF Calculator: Used in stock markets to calculate fees, taxes, and Margin Trading Facility (MTF) costs.
[ ] Inflation Calculator: Calculator the value of money adjusted for inflation
